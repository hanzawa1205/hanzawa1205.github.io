hexo搭建

Veröffentlicht am 2019-05-08

##hexo博客搭建
步骤1： 下载git
步骤2： 下载node.js
步骤3： 在桌面右键启动Git Bash后输入命令:
npm install -g hexo
步骤4： 创建一个’hexo’文件夹，在该文件夹中右键启动Git Bash后输入指令：
hexo init

此时hexo的静态文件等都已经加载到文件夹中。其中_config.yml文件可以修改博客的配置(标题，主题等等)
启动博客hexo g、hexo s
写博客：路径：\Hexo\source_posts

##部署到Github
步骤1： 在GitHub上创建仓库，仓库名为hanzawa1205.github.io
步骤2： 在_config.yml中配置

deploy

type: git
repo: 仓库地址

步骤3： 安装hexo-deployer-git自动部署发布工具，指令:npm install hexo-deployer-git --save
步骤4： 通过命令hexo clean && hexo g && hexo d发布到GitHub
最后输入http://https://hanzawa1205.github.io/测试访问部署完成的博客。
